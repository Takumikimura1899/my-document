## 更新の進め方

- 新しい知見やルールが生まれたら、関連ページを更新しチームに共有してください。
- 迷ったときは「はじめに」の各ページを基点に、詳細なカテゴリへ遷移できるようにしています。
- Pull Request では変更の背景と確認観点を明示し、レビュアーと合意を得た内容を残しましょう。
