## 対象範囲

- Astro / Starlight を中心としたサイト構築
- UI コンポーネント（React など）とデザインシステムの連携
- パフォーマンス最適化、アクセシビリティ、国際化対応

## 開発の基本姿勢

- コンポーネントは再利用性を意識して設計し、Storybook 等で可視化する
- デザインとの同期を保つため、デザインツールとドキュメントを往復する仕組みを整える
- ビルド時の警告や lint エラーは常に解消する
