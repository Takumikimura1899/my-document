---
title: スタイリングガイド
description: デザインシステムと整合したスタイルを実装するための指針です。
---

## 基本方針

- デザインシステムのトークン（色・タイポグラフィ・スペーシング）を参照する
- グローバルスタイルとコンポーネントスタイルを分離する
- レイアウトは CSS Grid / Flexbox を活用し、レスポンシブを意識する

## 運用

- 新しいトークンを追加する場合はデザインと合意し、ドキュメントを更新する
- カスタムスタイルが増えすぎた場合は共通化やリファクタリングを検討する
- パフォーマンス指標（CLS、LCPなど）をウォッチし、改善策を検討する

## チェックリスト

- コントラスト比が WCAG 基準を満たしている
- フォーカスインジケーターが消えていない
- モバイル・デスクトップ両方で崩れがない
