## 対象範囲

- API 設計とドメインモデリング
- データストアとキャッシュ戦略
- インフラ構成、デプロイ、監視

## 進め方の原則

- ドメインの境界を明確にし、依存を最小化する
- 自動テストと監視を標準装備として扱う
- 観測可能性（ログ、メトリクス、トレース）を重視する
